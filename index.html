Sobre o Projeto

O Comparador de Preços é uma ferramenta desenvolvida para ajudar os usuários a comparar os preços de itens de supermercado em diferentes estabelecimentos.

Foi criado em 2024 como parte de um projeto de desenvolvimento web.

Seu objetivo é facilitar a tomada de decisão dos consumidores, permitindo que eles encontrem os melhores preços para os produtos que desejam comprar.

Como Funciona

Para utilizar o Comparador de Preços, basta adicionar os itens desejados e os preços em cada mercado. Em seguida, clique no botão "Calcular" para ver os resultados.
